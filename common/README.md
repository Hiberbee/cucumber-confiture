## Installation

Maven `pom.xml`:
```xml
<dependency>
  <groupId>com.hiberbee</groupId>
  <artifactId>cucumber-common</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```

Gradle `build.gradle`:
```groovy
dependencies {
  testImplementation("com.hiberbee:cucumber-common:1.0.0-SNAPSHOT")
}
```
